# LVI-SAM_ROS2

本项目源于我的本科任务，当初曾对 `LVI-SAM` 进行了简单的 `ROS 2` 迁移，但是实际效果并不好，而且缺乏创新性。
最近回想起来，针对该项目进行完整的改进完成，并作为我的本科毕设。主要内容包括：

  1. 完成 `LVI-SAM` 向 `ROS 2` 的完整迁移
  2. 雷达部分利用 `Livox` 消息格式进行修改使用（主要是 `Livox` 雷达性价比足够高）。
  3. 相机部分使用更加现代化的 Vins 方法（最初版使用的 `Vins Mono` 模块过于糟糕了）。
  4. 对于雷达和相机部分均增加上重定位方式。
  5. 对雷达和相机部分算法引入 `CUDA` 实现加速。

## 2025. 11. 21

完成基于 `MID-360` 的 `Lio-SAM` 部分，并且增加需要初始位姿的点云重定位。事实证明，完全实现在已有点云地图中定位必须引入 `CUDA` 加速以保证 `KNN` 部分的实时性。
